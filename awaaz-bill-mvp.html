    Awaaz Bill 🎙

    Voice-powered billing for Indian kirana stores. Speak naturally in Hindi, English, or Hinglish to create bills instantly.

    How to use:
    1. Mic button dabayein (ya SPACE key)
    2. Item bolein: "Sugar ek kilo 50 rupaye"
    3. Multiple items: "Namak 24 rupaye aur doodh 30 rupaye"
    4. "Undo karo" — last item hata do
    5. "Print karo" — receipt print karein

    This is an MVP prototype. Voice works best on Chrome + Android.
    API key is stored in your browser only.
